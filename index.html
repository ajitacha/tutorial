What is C++

C++ is a general purpose, case-sensitive, free-form programming language that supports object-oriented, procedural and generic programming.

C++ is a middle-level language, as it encap sulates both high and low level language features.

Object Oriented Concepts

C++ supports the object-oriented programming, the four major pillar of object oriented programming used in C++ are:


  • Inheritance
  • Polymorphism
  • Encapsulation
  • Abstraction

C++ if-else

In C++ programming, if statement is used to test the condition. There are various types of if statements in C++.


  • if statement
  • if-else statement
  • nested if statement
  • if-else-if ladder

C++ IF Statement

The C++ if statement tests the condition. It is executed if condition is true.

if(condition){
//code to be executed
}
